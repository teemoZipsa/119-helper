# 119 Helper - 추가 기획 및 아이디어 저장소

## 1. 🚁 [소방 드론/항공] 비행 안전 지수 (Aviation Index)
**목적:** 현장 정찰용 소방 드론 및 구조 헬기의 안전 이륙 여부를 직관적으로 판단할 수 있는 비행 안전 신호등 기능

**핵심 아이디어:**
- 기상청 실황풍속 또는 행정안전부 풍속 API(`DSSP-IF-00028`)를 수신
- **판단 로직:** 
  - `풍속 < 7m/s`: 🟢 비행 양호
  - `7m/s <= 풍속 < 10m/s`: 🟡 주의 (돌풍 감시)
  - `풍속 >= 10m/s`: 🔴 ⛔ 비행 위험 (비행 자제 권고)
- **UI 구현 모델:** 
  - 나중에 신설될 `[항공 / 드론]` 전용 탭에 거대한 계기판 형식으로 제공.
  - 대시보드에는 위젯 형태로 조그맣게 `드론: 🟢 양호` 또는 `드론: 🔴 불가` 뱃지를 달아준다.
- **기대 효과:** 현장 지휘관 및 조종사가 드론이나 헬기를 운용하기 전 즉각적인 안전 결정을 내릴 수 있도록 보조.
